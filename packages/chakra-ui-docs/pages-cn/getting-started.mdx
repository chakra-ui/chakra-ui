# 开始上手

安装查克拉UI和其依赖包。

```bash
npm install @chakra-ui/core @emotion/core @emotion/styled emotion-theming
```

创建一个主题对象，包括自定义颜色、字体和布局。
查克拉UI自带默认主题，所以这一步为自选。

```js
// theme.js示例
export default {
  breakpoints: ["30em", "48em", "62em", "80em"],
  fonts: {
    heading: '"Avenir Next", sans-serif',
    body: "system-ui, sans-serif",
    mono: "Menlo, monospace",
  },
  fontSizes: {
    xs: "0.75rem",
    sm: "0.875rem",
    md: "1rem",
    lg: "1.125rem",
    xl: "1.25rem",
    "2xl": "1.5rem",
    "3xl": "1.875rem",
    "4xl": "2.25rem",
    "5xl": "3rem",
    "6xl": "4rem",
  },
};
```

你也可以扩展主题，加入自定义的颜色、字体等。

```js
import { theme } from "@chakra-ui/core";

// 比如你想加入自定义颜色
const customTheme = {
  ...theme,
  colors: {
    ...theme.colors,
    brand: {
      900: "#1a365d",
      800: "#153e75",
      700: "#2a69ac",
    },
  },
};
```

用`ThemeProvider`为你的应用加入主题，主题对象作为属性传入`ThemeProvider`。

```jsx live=false
import React from "react";
import { ThemeProvider } from "@chakra-ui/core";

// 在应用根文件使用
<ThemeProvider theme={customTheme}>{props.children}</ThemeProvider>;
```

## 使用组件

按名称输入`@chakra-ui/core`中的组件,
示例如下。

```js
import { Button } from "@chakra-ui/core";
```

## 注入全局样式

有时你可能需要重置css样式。查克拉UI输出的`CSSReset`可以移除浏览器默认样式。
这是受[Tailwind的预检](https://unpkg.com/tailwindcss@1.1.2/dist/base.css)所启发。

> 🚨 我们强烈建议在应用根文件加入`CSSReset`，
> 以确保所有组件正确运行。

```jsx live=false
// import customTheme from "./theme"

function App({ children }) {
  return (
    <ThemeProvider theme={customTheme}>
      <CSSReset />
      {children}
    </ThemeProvider>
  );
}
```

## 加入图标

你可以用标准库[react-icons](https://react-icons.netlify.com)中的自定义图标，
你也可以在主题对象中加入图标`path`。

在主题对象中加图标，只需如下面示例定义`icons`键。

```js
// theme.js
export default {
  icons: {
    // 加入查克拉图标
    ...theme.icons,
    // 你的自定义图标
    search: {
      // svg用<path/>或<g> 注意`fill=currentColor`的使用
      path: (
        <path
          fill="currentColor"
          d="M23.384,21.619,16.855,15.09a9.284,9.284,0,1,0-1.768,1.768l6.529,6.529a1.266,1.266,0,0,0,1.768,0A1.251,1.251,0,0,0,23.384,21.619ZM2.75,9.5a6.75,6.75,0,1,1,6.75,6.75A6.758,6.758,0,0,1,2.75,9.5Z"
        />
      ),
      // 这是图标的视框
      viewBox: "0 0 24 24",
    },
  },
};
```

加入图标后，你可以如下面示例在`Icon`组件中使用：

```jsx live=true
<Icon name="search" color="red.500" />
```
