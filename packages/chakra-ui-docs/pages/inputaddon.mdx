# Input Addon

Input Addon is a component used to extend form controls by adding text or icons
before, after, or on both sides of any text-based [Input](/input).

## Installation

```bash
npm install @chakra-ui/core
```

then import InputAddon

```js
import { InputAddon } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputAddon placement="left">+234</InputAddon>
      <Input roundedLeft="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

```jsx
function Example() {
  return (
    <InputGroup size="lg">
      <InputAddon placement="right">
        <Icon name="search" />
      </InputAddon>
      <Input roundedRight="0" placeholder="Search" />
    </InputGroup>
  );
}
```

## Props

| Name        | Type              | Default | Description                                                                                                                                      |
| ----------- | ----------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `size`      | `sm`, `md`, `lg`  | `md`    | The size of the `InputAddon`.                                                                                  |
| `children`  | `React.ReactNode` | `false` | The children of the `InputAddon`.                                                                                  |
| `placement` | `left`, `right`   | `left`  | The position the addon should appear relative to the `Input`. We added `InputLeftAddon` and `InputRightAddon` so you might not need to pass this. |
