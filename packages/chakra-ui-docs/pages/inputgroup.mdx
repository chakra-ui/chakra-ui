# Input Group

The InputGroup is a wrapper component used for the composition of input elements
with addons, icons or other elements on either side of it.

## Installation

```bash
npm install @chakra-ui/core
```

## Usage

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftAddon>+234</InputLeftAddon>
      <Input roundedLeft="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

## Compostion

Use the `InputGroup` to compose form input components by placing an
[addon-on](/inputaddon) or [Button](/button) on either side of an
[Input](/input). You may also place one on both sides of an input as well as
have a combination of [addon-on](/inputaddon), [Icon](/icon) or
[Button](/button).

### With left addon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftAddon>+234</InputLeftAddon>
      <Input roundedLeft="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With right addon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputRightAddon>.com</InputRightAddon>
      <Input roundedRight="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With left icon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftElement>
        <Icon name="phone" size="1em" />
      </InputLeftElement>
      <Input placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With right icon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputRightElement>
        <Icon name="check" size="1em" />
      </InputRightElement>
      <Input placeholder="Welcome" />
    </InputGroup>
  );
}
```

## Props

| Name       | Type              | Default | Description                                                     |
| ---------- | ----------------- | ------- | --------------------------------------------------------------- |
| `size`     | `sm`, `md`, `lg`  | `md`    | The size of the input component composed with the `InputGroup`. |
| `children` | `React.ReactNode` | `false` | The children of the `InputGroup` e.g `InputLeftAddon`, `Input`. |
