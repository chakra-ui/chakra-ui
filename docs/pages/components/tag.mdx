# Tag

Tag component is used for items that need to be labeled, categorized, or
organized using keywords that describe them.

## Import

Chakra UI exports 4 Tag related components:

- **Tag**: The wrapper for all the tag elements
- **TagLabel**: The label for tag's text content.
- **TagIcon**: The icon for the tag
- **TagCloseButton**: The close button for the tag.

```js
import { Tag, TagIcon, TagLabel, TagCloseButton } from "@chakra-ui/core"
```

## Usage

```jsx
<Tag>Sample Tag</Tag>
```

```jsx
<Stack spacing={4} isInline>
  {["sm", "md", "lg"].map((size) => (
    <Tag size={size} key={size} colorScheme="gray">
      Gray
    </Tag>
  ))}
</Stack>
```

## With left icon

```jsx
<Stack spacing={4} isInline>
  {["sm", "md", "lg"].map((size) => (
    <Tag size={size} key={size} colorScheme="cyan">
      <TagIcon icon="add" size="12px" />
      <TagLabel>Green</TagLabel>
    </Tag>
  ))}
</Stack>
```

## With right icon

```jsx
<Stack spacing={4} isInline>
  <Tag colorScheme="cyan">
    <TagLabel>Green</TagLabel>
    <TagIcon icon="check" size="12px" />
  </Tag>

  {/* You can also use custom svg icons */}
  <Tag colorScheme="teal">
    <TagLabel>Green</TagLabel>
    <TagIcon icon={MdSettings} />
  </Tag>
</Stack>
```

## With close button

```jsx
<Stack spacing={4} isInline>
  {["sm", "md", "lg"].map((size) => (
    <Tag
      size={size}
      key={size}
      borderRadius="full"
      variant="solid"
      colorScheme="cyan"
    >
      <TagLabel>Green</TagLabel>
      <TagCloseButton />
    </Tag>
  ))}
</Stack>
```

## With custom element

```jsx
<Tag colorScheme="red" borderRadius="full">
  <Avatar
    src="https://bit.ly/sage-adebayo"
    size="xs"
    name="Segun Adebayo"
    ml={-1}
    mr={2}
  />
  <TagLabel>Segun</TagLabel>
</Tag>
```

## Props

| Name          | Type                         | Default | Description                             |
| ------------- | ---------------------------- | ------- | --------------------------------------- |
| `variant`     | `solid`, `subtle`, `outline` | `solid` | The variant style of the tag component. |
| `size`        | `sm`, `md`, `lg`             | `md`    | The size of the tag component.          |
| `colorScheme` | `string`                     |         | The color scheme of the tag variant.    |
