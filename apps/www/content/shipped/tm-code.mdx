Célio Garcia is the CEO of [TM Code](https://code.toquemedia.net), an
agent-first IDE that puts Chat and Terminal mode in one place. It's made for
developers, vibe coders, DevOps, and IT enthusiasts. He built the interface with
Chakra UI.

Before TM Code, Célio worked mostly in native development with Jetpack Compose
and SwiftUI. So when he tried Chakra, it felt familiar. **"Chakra UI reminds me
a lot of Jetpack Compose and SwiftUI,"** he says. **"Both are great frameworks.
That's why I loved Chakra UI on day one."** The declarative style was something
he already knew, so he didn't spend long getting comfortable with it.

The biggest win was the UI. When we asked what Chakra let him do better than he
expected, he said _"definitely user interfaces."_ For an IDE, that's the part
users sit in all day.

It wasn't all smooth. Chakra v3's new theming tripped him up at first, but he
worked through it and shipped. TM Code is live now at
[code.toquemedia.net](https://code.toquemedia.net).

As for whether other developers should reach for Chakra, Célio keeps it short:
_"Go on, and thank me later."_
