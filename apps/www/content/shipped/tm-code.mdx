## 👋🏽 A bit of context

[TM Code](https://code.toquemedia.net) is an agent first IDE where I describe
what I want built, and the agent writes the code, shows the diff, opens a live
preview, and deploys the project.

I'm Célio Garcia, the CEO and founder of TM Code. Before building TM code, I was
familiar with mobile UI frameworks like Jetpack Compose and SwiftUI. I
understood how those tools worked and the approach behind them, so finding the
right UI library for my IDE was one of the first decisions I had to make.

That search led me to Chakra UI.

## 💙 Fav thing about Chakra UI

With Chakra, I didn't need to spend time learning a new way of building
interfaces. Instead, I could focus on the product itself. What Chakra let me do
faster or better than I expected was simple: **"Definitely user interfaces."**

Chakra v3's theming was the early hurdle. The v3 release changed how themes are
defined, but once I understood the new approach, Chakra carried the rest of the
interface work through to launch at
[code.toquemedia.net](https://code.toquemedia.net). Nothing else slowed me down.
