## Overview

The Chakra UI 3.0 preview release is now available. This early preview is
intended to gather feedback from the community before the final release.

The key features of Chakra 3.0 are:

- Next.js RSC
- Supercharged components from Ark UI
- New theming API
- New composable components
- New layout components
- New hooks
- New utilities

## Try it out

Try it out and give us your feedback.

```bash
npm install @chakra-ui/react@next
```
