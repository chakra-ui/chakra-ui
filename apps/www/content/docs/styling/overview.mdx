## Nested Styles

For type safety, Chakra requires that you use the `&` operator to nest styles.

```tsx
const Card = chakra("div", {
  base: {
    bg: "white",
    borderRadius: "md",
    boxShadow: "md",
    "& svg": {
      color: "red.500",
    },
  },
})
```
