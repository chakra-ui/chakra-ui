## Setup

If you don't already have the snippet, run the following command to add the
`select` snippet

```sh
chakra snippet add select
```

The snippet includes a closed component composition for the `Select` component.

## Usage

```jsx
import {
  SelectContent,
  SelectItem,
  SelectLabel,
  SelectRoot,
  SelectTrigger,
  SelectValueText,
} from "@/components/ui/select"
```

```jsx
<SelectRoot>
  <SelectLabel />
  <SelectTrigger>
    <SelectValueText />
  </SelectTrigger>
  <SelectContent>
    <SelectItem />
  </SelectContent>
</SelectRoot>
```
