## Overview

A date picker component designed for selecting date ranges. It supports various
selection granularities including full date ranges, month ranges, and year
ranges. Features like min/max constraints, preset ranges, and multiple selection
modes make it flexible for different use cases such as booking periods,
reporting intervals, or event date ranges.

## Installation

```tsx
import { DatePicker } from "@chakra-ui/react"
```

## Basic Usage

```tsx
<DatePicker.Root>
  {data.map((item) => (
    <DatePicker.Item key={item.label}>
      <DatePicker.ItemLabel>{item.label}</DatePicker.ItemLabel>
      <DatePicker.ItemValue>{item.value}</DatePicker.ItemValue>
    </DatePicker.Item>
  ))}
</DatePicker.Root>
```
