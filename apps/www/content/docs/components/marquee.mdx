## Usage

```js
import { Marquee } from "@chakra-ui/react"
```

```jsx
<Marquee.Root>
  <Marquee.Viewport>
    <Marquee.Content>
      <Marquee.Item></Marquee.Item>
    </Marquee.Content>
  </Marquee.Viewport>
</Marquee.Root>
```

## Guide

### Anatomy

The Marquee component is composed of several parts:

- **Root**: The main container for the marquee.
- **Viewport**: The visible area through which the content scrolls.
- **Content**: The scrolling content inside the marquee.
- **Item**: Each item or child inside the marquee content.
- **Edge**: Optional gradient edge for visual effect.
- **Control**: Optional controls for programmatic interaction.

### Tips & Best Practices

- Use the `autoFill` prop to automatically repeat items for seamless scrolling.
- Use the `reverse` or `side` prop for direction/orientation.
- Use `pauseOnInteraction` to pause on hover or focus.
- Use `Edge` for gradient fade effects at the start/end.
- Use `loopCount` and event handlers for finite or controlled loops.
