## Overview

A controlled date picker component where the selected date value is managed by
React state. This gives you full control over the picker's behavior and allows
you to synchronize it with your application's state management.

## Installation

```tsx
import { DatePicker } from "@chakra-ui/react"
```

## Basic Usage

```tsx
<DatePicker.Root>
  {data.map((item) => (
    <DatePicker.Item key={item.label}>
      <DatePicker.ItemLabel>{item.label}</DatePicker.ItemLabel>
      <DatePicker.ItemValue>{item.value}</DatePicker.ItemValue>
    </DatePicker.Item>
  ))}
</DatePicker.Root>
```
