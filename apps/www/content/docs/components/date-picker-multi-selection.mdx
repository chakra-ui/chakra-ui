## Overview

A date picker component that allows users to select multiple dates at once.
Perfect for scenarios like booking date ranges, selecting multiple event dates,
or any use case requiring multiple date selections. Includes support for
displaying multiple calendar months simultaneously for better usability.

## Installation

```tsx
import { DatePicker } from "@chakra-ui/react"
```

## Basic Usage

```tsx
<DatePicker.Root>
  {data.map((item) => (
    <DatePicker.Item key={item.label}>
      <DatePicker.ItemLabel>{item.label}</DatePicker.ItemLabel>
      <DatePicker.ItemValue>{item.value}</DatePicker.ItemValue>
    </DatePicker.Item>
  ))}
</DatePicker.Root>
```
