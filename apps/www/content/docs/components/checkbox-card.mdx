## Setup

If you don't already have the snippet, run the following command to add the
`checkbox-card` snippet

```sh
chakra snippet add checkbox-card
```

The snippet includes a closed component composition for the `CheckboxCard`
component.

## Usage

```jsx
import {
  CheckboxCardAddon,
  CheckboxCardControl,
  CheckboxCardLabel,
  CheckboxCardRoot,
} from "@/components/ui/checkbox-card"
```

```jsx
<CheckboxCardRoot>
  <CheckboxCardControl>
    <CheckboxCardLabel />
  </CheckboxCardControl>
  <CheckboxCardAddon />
</CheckboxCardRoot>
```
