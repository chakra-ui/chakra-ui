## Usage

```jsx
import { Splitter } from "@chakra-ui/react"
```

```jsx
<Splitter.Root>
  <Splitter.Panel id="a">Panel A</Splitter.Panel>
  <Splitter.ResizeTrigger id="a:b" aria-label="Resize panels" />
  <Splitter.Panel id="b">Panel B</Splitter.Panel>
</Splitter.Root>
```

### Store

Use the `RootProvider` with the `useSplitter` hook to manage and control the
splitter state from outside the component.

> Use `RootProvider + useSplitter` or `Root`, not both.

<ExampleTabs name="splitter-with-store" />

### Collapsible Panels

Panels can be made collapsible by setting the `collapsible` and `collapsedSize`
properties on a panel in the `panels` array. This allows users to collapse
panels to a defined minimum size.
