## Setup

If you don't already have the snippet, run the following command to add the
`clipboard` snippet

```sh
chakra snippet add clipboard
```

The snippet includes a closed component composition for the `Clipboard`
component.

## Usage

```jsx
import {
  ClipboardButton,
  ClipboardIconButton,
  ClipboardInput,
} from "@/components/ui/clipboard"
```

```jsx
<ClipboardRoot value="...">
  <ClipboardButton />
  <ClipboardIconButton />
  <ClipboardInput />
</ClipboardRoot>
```
