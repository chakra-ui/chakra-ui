## Setup

If you don't already have the snippet, run the following command to add the
`menu` snippet

```sh
chakra snippet add menu
```

The snippet includes a closed component composition for the `Menu` component.

## Usage

```jsx
import {
  MenuContent,
  MenuItem,
  MenuRoot,
  MenuTrigger,
} from "@/components/ui/menu"
```

```jsx
<MenuRoot>
  <MenuTrigger />
  <MenuContent>
    <MenuItem value="..." />
  </MenuContent>
</MenuRoot>
```
