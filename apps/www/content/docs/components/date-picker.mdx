## Usage

```tsx
import { DatePicker } from "@chakra-ui/react"
```

```tsx
<DatePicker.Root>
  <DatePicker.Label />
  <DatePicker.Control>
    <DatePicker.Input />
    <DatePicker.IndicatorGroup>
      <DatePicker.Trigger>
        <LuCalendar />
      </DatePicker.Trigger>
    </DatePicker.IndicatorGroup>
  </DatePicker.Control>
  <Portal>
    <DatePicker.Positioner>
      <DatePicker.Content>
        <DatePicker.Header />
        <DatePicker.DayView />
        <DatePicker.MonthView />
        <DatePicker.YearView />
      </DatePicker.Content>
    </DatePicker.Positioner>
  </Portal>
</DatePicker.Root>
```

The simplest implementation of DatePicker allows users to select a single date
with minimal configuration.

want to match your design system's button styles.

### Controlled

Use 'Datepicker context' to control the date picker value programmatically.

<ExampleTabs name="date-picker-controlled" />

**Why use controlled components?**

- Sync date selection with other form fields
- Trigger side effects when dates change
- Implement custom validation logic
- Reset or update dates programmatically
