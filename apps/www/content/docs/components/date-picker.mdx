## Overview

The Date Picker component provides a user-friendly interface for selecting
dates. It supports various configurations including inline display, custom
formatting, localization, and date availability constraints.

## Installation

```tsx
import { DatePicker } from "@chakra-ui/react"
```

## Basic Usage

A simple date picker that allows users to select a single date.

<ExampleTabs name="date-picker-basic" />

```tsx
<DatePicker.Root>
  {data.map((item) => (
    <DatePicker.Item key={item.label}>
      <DatePicker.ItemLabel>{item.label}</DatePicker.ItemLabel>
      <DatePicker.ItemValue>{item.value}</DatePicker.ItemValue>
    </DatePicker.Item>
  ))}
</DatePicker.Root>
```
