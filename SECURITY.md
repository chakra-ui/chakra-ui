# Security

We take security seriously. If you believe you've found a vulnerability, please
report it responsibly.

## Reporting a Vulnerability

**Do not report security vulnerabilities in public issues, discussions, or pull
requests.**

Report privately via
[GitHub Security Advisories](https://github.com/chakra-ui/chakra-ui/security/advisories/new).

Include as much of the following as you can:

- Type of issue (e.g. XSS, injection)
- Affected package and version
- Steps to reproduce
- Impact and possible exploit

We'll acknowledge your report and keep you updated on the fix.

## Supported Versions

We release security updates for the latest major version. Upgrade to receive
patches.
