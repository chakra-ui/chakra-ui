### Minor Changes

#### @chakra-ui/styled-system@2.5.0:

- Bring back `isTruncated` to solve some edge cases with `noOfLines={1}`

#### @chakra-ui/gatsby-plugin@3.1.0:

- Add option to use `ChakraBaseProvider` instead of `ChakraProvider`. This will
  only apply theme tokens initially and not default theming for components.

### Patch Changes

#### @chakra-ui/card@2.1.3:

- Fix theme being overriden

#### @chakra-ui/icons@2.0.14:

- Add `ReactIcon` to align with Figma UI Kit

#### @chakra-ui/skeleton@2.0.20:

- Add support for `fitContent` prop.

  When set to `true`, the `Skeleton` will take the width of its children.

  ```jsx live=false
  <Skeleton fitContent>
    <p>Hello World</p>
  </Skeleton>
  ```

- Fix theme being overidden in dark mode

#### @chakra-ui/slider@2.0.16:

- Expose `RangeSliderState` and `RangeSliderActions` types to improve
  documentation

#### @chakra-ui/table@2.0.13:

- Added the `layout` prop to set the `tableLayout` CSS property.

#### @chakra-ui/cli@2.2.1:

- Support TS Config paths more completely
