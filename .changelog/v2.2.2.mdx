### Minor Changes

#### @chakra-ui/color-mode@2.1.0:

- Check if the MediaQueryList object supports the addEventListener() method,
  else fallback to the legacy .addListener() method.

#### @chakra-ui/modal@2.1.0:

- Export `useDrawerContext` hook

#### @chakra-ui/system@2.2.0:

- Extract and export `getToken` function from `useToken`
- Improve parameter types for `useToken`

### Patch Changes

- Remove internal usage of createBreakpoints function

#### @chakra-ui/checkbox@2.1.1:

- fixed issue where switch cannot be toggled when text is selectedd

#### @chakra-ui/descendant@3.0.3:

- Apply type constraints to `useDescendants` hook for improved TypeScript 4.8
  support

#### @chakra-ui/layout@2.0.3:

- SimpleGrid: Fix issue where `minChildWidth` doesn't work with size tokens

#### @chakra-ui/radio@2.0.3:

- Fix issue where radio doesn't show focus when interacting with keyboard

#### @chakra-ui/skeleton@2.0.7:

- extend doc for `fadeDuration` prop
- Remove internal usage of createBreakpoints function

#### @chakra-ui/toast@3.0.0:

- Move `@chakra-ui/system` to peer dependency

#### @chakra-ui/tooltip@2.0.3:

- Avoid rendering multiple tooltips so that they are not appeared after opening
  a modal

#### @chakra-ui/cli@2.1.1:

- Add --no-format option
- Fix issue where CLI crashed due to lodash throttle dependency
