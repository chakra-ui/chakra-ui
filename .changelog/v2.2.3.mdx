### Minor Changes

#### @chakra-ui/layout@2.1.0:

- Add new highlight text component to emphasize any part of a string with the
  `mark` tag.

  ```jsx live=false
  <Text fontWeight="semibold">
    <Highlight
      query={"with speed"}
      styles={{ px: "2", py: "1", rounded: "full", bg: "teal.100" }}
    >
      Create accessible React apps with speed
    </Highlight>
  </Text>
  ```

### Patch Changes

#### @chakra-ui/accordion@2.0.5, @chakra-ui/alert@2.0.3, @chakra-ui/avatar@2.0.4, @chakra-ui/breadcrumb@2.0.3, @chakra-ui/editable@2.0.3, @chakra-ui/form-control@2.0.3, @chakra-ui/menu@2.0.5, @chakra-ui/number-input@2.0.3, @chakra-ui/progress@2.0.4, @chakra-ui/slider@2.0.3, @chakra-ui/stat@2.0.3, @chakra-ui/table@2.0.3, @chakra-ui/tabs@2.0.5, @chakra-ui/tag@2.0.3:

- Export `useStyles` equivalent for multipart component styles. Accordion
  exports `useAccordionStyles`, Alert exports `useAlertStyles`, and so on.

#### @chakra-ui/react-env@2.0.3, @chakra-ui/media-query@3.1.2, @chakra-ui/portal@2.0.3:

- Fix hydration issues when using `Suspense`

#### @chakra-ui/image@2.0.4, @chakra-ui/modal@2.1.1:

- Add `displayName` to components using `forwardRef`

#### @chakra-ui/popover@2.0.3:

- Export `useStyles` equivalent for multipart component styles. Accordion
  exports `useAccordionStyles`, Alert exports `useAlertStyles`, and so on.
- Add `displayName` to components using `forwardRef`
- Fix issue where popover opens when mouse click opens the popover and trigget
  is set to `hover`.

#### @chakra-ui/theme@2.1.2:

- Add border radius of zero to the default full size modal

#### @chakra-ui/transition@2.0.3:

- Fix emitted types that were incompatible with latest `framer-motion` release

#### @chakra-ui/utils@2.0.3:

- Fix `TypeError` in `getCSSVar` when the theme is an empty object
- Fix issue where `getFocusables` causes excessive repaint due to computed style
  calls

#### @chakra-ui/babel-plugin@1.0.4, @chakra-ui/cra-template@2.0.3, gatsby-starter-default@1.0.9:

- Update ESlint packages and config

#### @chakra-ui/cli@2.1.2:

- Fix importing of the default exported theme
