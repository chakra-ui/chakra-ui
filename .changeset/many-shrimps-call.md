The official Storybook Addon for Chakra UI.

```sh
yarn add -D @chakra-ui/storybook-addon
```

```sh
npm i -D @chakra-ui/storybook-addon
```

Add the addon to your configuration in `.storybook/main.js`:

```js
module.exports = {
  addons: ["@chakra-ui/storybook-addon"],
}
```
