## Import

```js
import { Divider } from "@chakra-ui/core"
```

## Usage

The `Divider` displays a thin horizontal or vertical line, and renders a `hr` tag.

```jsx
<Divider />
```

### Divider Orientation

Pass the `orientation` prop and set it to either `horizontal` or `vertical`.

```jsx
<Divider orientation="horizontal" />
```

If the vertical orientation is used, make sure that the parent element is
assigned a height.

```jsx
<Center height="50px">
  <Divider orientation="vertical" />
</Center>
```

## Composition

```jsx
<Stack direction="row" h="100px" p={4}>
  <Divider orientation="vertical" />
  <Text>Chakra UI</Text>
</Stack>
```

## Props

| Name          | Type                     | Default      | Description                    |
| ------------- | ------------------------ | ------------ | ------------------------------ |
| `orientation` | `horizontal`, `vertical` | `horizontal` | The orientation of the divider. |
