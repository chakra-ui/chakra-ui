package: '@chakra-ui/progress'
description:
  Progress is used to display the progress status for a task that takes a long
  time or consists of several steps.
---

## Import

```js
import { Progress } from '@chakra-ui/react'
```

## Usage

```jsx
<Progress.Root value={80}>
  <Progress.Track>
    <Progress.FilledTrack />
  </Progress.Track>
  <Progress.ValueText />
</Progress.Root>
```

### Progress with Stripe

You can add `hasStripe` prop to any progress bar to apply a stripe via a CSS
gradient over the progress barâ€™s background color.

```jsx
<Progress.Root hasStripe value={64}>
  <Progress.Track>
    <Progress.FilledTrack />
  </Progress.Track>
  <Progress.ValueText />
</Progress.Root>
```

### Progress sizes

There are two ways you can increase the height of the progress bar:

- You can add `size` prop to increase the height of the progress bar.
- You can also use the `height` prop to manually set a height.

```jsx
<Stack spacing={5}>
  <Progress.Root colorScheme='green' size='sm' value={20}>
    <Progress.Track>
      <Progress.FilledTrack />
    </Progress.Track>
    <Progress.ValueText />
  </Progress.Root>
  <Progress.Root colorScheme='green' size='md' value={20}>
    <Progress.Track>
      <Progress.FilledTrack />
    </Progress.Track>
    <Progress.ValueText />
  </Progress.Root>
  <Progress.Root colorScheme='green' size='lg' value={20}>
    <Progress.Track>
      <Progress.FilledTrack />
    </Progress.Track>
    <Progress.ValueText />
  </Progress.Root>
  <Progress.Root colorScheme='green' height='32px' value={20}>
    <Progress.Track>
      <Progress.FilledTrack />
    </Progress.Track>
    <Progress.ValueText />
  </Progress.Root>
</Stack>
```

### Progress color scheme

You can add `colorScheme` prop to any progress bar to apply any color that
exists in the `theme`.

```jsx
<Progress.Root value={20} size='xs' colorScheme='pink'>
  <Progress.Track>
    <Progress.FilledTrack />
  </Progress.Track>
  <Progress.ValueText />
</Progress.Root>
```

### Animated Progress

To show an animated progress, pass the `isIndeterminate` prop.

```jsx
<Progress.Root size='xs' isIndeterminate>
  <Progress.Track>
    <Progress.FilledTrack />
  </Progress.Track>
  <Progress.ValueText />
</Progress.Root>
```

## Accessibility

- `Progress.FilledTrack` has a `role` set to `progressbar` to denote that it is
  a progress.
- `Progress.FilledTrack` has `aria-valuenow` set to the percentage completion
  value passed to the component, to ensure the progress percent is visible to
  screen readers.
