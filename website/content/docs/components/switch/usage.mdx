The `Switch` component is used as an alternative for the
[Checkbox](/docs/components/form/checkbox) component. You can switch between
enabled or disabled states.

<carbon-ad></carbon-ad>

`Switch` must always be accompanied by a label, and follows the same keyboard
workflow as a `Checkbox`.

## Import

```js
import { Switch } from '@chakra-ui/react'
```

## Usage

```jsx
<Field.Root display='flex' alignItems='center'>
  <Field.Label htmlFor='email-alerts' mb='0'>
    Enable email alerts?
  </Field.Label>
  <Switch.Root id='email-alerts'>
    <Switch.Track>
      <Switch.Thumb />
    </Switch.Track>
  </Switch.Root>
</Field.Root>
```

## Changing the size

Pass the `size` prop to change the size of the `Switch`.

```jsx
<Switch.Root size='sm'>
  <Switch.Track>
    <Switch.Thumb />
  </Switch.Track>
</Switch.Root>
```

## Changing the color palette

Set the `colorPalette` prop to change the color of the `Switch`.

```jsx
<Switch.Root size='sm' colorPalette='teal'>
  <Switch.Track>
    <Switch.Thumb />
  </Switch.Track>
</Switch.Root>
```
