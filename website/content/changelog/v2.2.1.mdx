## Minor Changes

### Toast `2.1.0`

- Add support for `portalProps` in toast provider. When using with the
  `ChakraProvider`, this can be configure in the `toastOptions`

## Patch Changes

### System `2.1.3`

- Fixed a regression where `useStyles` and `StylesProvider` was removed from
  `@chakra-ui/system`
